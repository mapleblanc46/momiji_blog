個人ブログをやるぞ！
Digital Garden、私が好きにいじれる電子の庭

1年ほど前にHugo+GitHub Pagesで個人ブログを作ってみたのだが、記事を書く習慣がつかず放置。半年くらい前から自宅サーバーを始めて、Obsidianを使う環境を整備してからメモを取る習慣がついてきて、記事にするネタも貯まってきた。そろそろやり直してもいい頃合いでしょう。

# ブログコンセプト

- 私のインターネットでの活動の本拠地としたい
	- とりあえず個人活動で試したこととか、得た学び・経験を出す場所
	- 通過したTRPGシナリオ・回したシナリオのメモ、イラストetc...も出したい
	- いずれは個人ドメインを持って色々ホスティングとかしてみたいけど、色々な対策やるの個人じゃだるそうなんだよなぁ まぁおいおい考える
- 往年の個人ブログのようでありたい
	- ポート開放できない！MODの競合が解決できない！といったときに調べて出てきたあの個人ブログたちの心強さったらないね
	- 記事本文の横にインデックスが表示されるみたいなスタイルが読みやすくていいなと感じる
		- QiitaとかZennとかのイメージ
- 検索機能
	- タグ分類ができたら嬉しい
	- 全文検索はなくてもいいかな
- 文章を書く練習がしたい。いつもはX(旧Twitter)でオタクの鳴き声を上げてるだけなので、このままでは人の形を保てなくなりそう
	- まずは作り始めて、アウトプットすることを習慣づける
	- SSGを利用してサクッとブログを作成したい
		- (いずれは)HTML, CSS, JSを理解して自分好みのサイトを作り上げたい

# 技術構成検討

個人ブログとしてなのでSSG+ホスティングサイトで十分だと判断した。
調べる中で候補に挙がったのはこの子達。

- ホスティング
    - AWS Lightsail
	    - 定額でサーバーからいじることができる
	    - 自分でいろいろ設定するのは勉強になりそうだけど手間でもある
    - Cloudflare Pages/Workers
	    - もろもろのセキュリティが無料プランでついてくる
	    - ドメイン購入もcloudflareを利用すれば、一つのダッシュボードで全部管理できる
    - GitHub Pages
	    - 無料でドメインまでつけて公開できる
	    - リポジトリでちょっと設定すればすぐに公開できる
		    - 昔ちょろっと触っていたので↑二つに比べて多少分かる
- SSG
    - Zola
	    - Rust製でビルドが速いらしい
	    - テンプレートエンジンがjinja2風らしい
		    - さぷーさんの動画でちょろーーっと触ったのでなんとなく親近感がある
    - Hugo
	    - GitHub Pagesと同じく昔ちょろっと触っていた
    - Astro
    - Next.js
	    - この二つはSSGというより、SSGとしても使えるフレームワーク？
	    - ECサイトなんかを作るのにも使えたりするらしい？
	    - 後々他のことにも手を出したくなった時にはいいかも
	    - その分学習コストが高そう

# 結論

- ホスティング
	- GitHub Pages
		- ドメインがついてくる
		- とりあえずすぐ公開したかった
		- GitHub Actionsを使用して直接デプロイできる
			- GitHub Actionsの勉強も併せてしてみたい
- SSG
	- Zola
		- Rust製というのがいい、Rustの勉強もしたいからやる気が出る
		- テーマの多さはHugoが魅力的だがモチベ優先

ホスティングに関しては後々Cloudflare Pagesに移行したいと思ってるけど、ブログを習慣化したいからとりあえず公開優先でGitHub  Pagesを選択。
SSGに関しては、「Rust、かっこいい……」というフィーリングでほぼ決め打ち。いや、モチベって大事ですやん……検討の意味とは。

絵も描きたいし、ゲームもしたいし、TRPGもしたいし、自宅サーバーもいじりたいし、インフラの上で動くサービス達のことも知りたいし、やりたいことが多すぎる。
デジタルなことはできる限り衆目に晒すようにして自身の成長、モチベにつなげたい。
これからがんばっていきましょい。
